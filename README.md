# Book Buzz 📚

Book Buzz is a Library Store Management System built to manage books, members, and library operations efficiently.  
The project provides a simple interface for managing book records and improving the overall library experience.

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- Material UI
- JavaScript
- Firebase

## 📂 Project Purpose

This project is developed to practice frontend development concepts, state management, CRUD operations, and building a real-world library management application.
